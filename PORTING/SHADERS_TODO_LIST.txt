2. FILTER MODES (SHADER-BASED)
================================================================================

  [100%] Point filtering (GL)                     — GL_NEAREST on emulator texture
  [100%] Bilinear filtering (GL)                  — GL_LINEAR on emulator texture
  [100%] Sharp bilinear                           — Fragment shader: snap to nearest
                                                    source pixel center, sharpness-
                                                    controlled bilinear blend via
                                                    kSFX_Sharp feature flag
  [100%] Bicubic (horizontal pass)                — Separable cubic filter via lookup
                                                    texture. Renders to mBicubicFBO.
  [100%] Bicubic (vertical pass)                  — Same filter applied vertically
                                                    from mBicubicFBO to mBicubicFBO2
  [100%] Bicubic lookup texture generation        — VDDisplayCreateBicubicTexture()
                                                    output uploaded to GL texture
  [100%] Filter sharpness levels (5)              — ATUIGetViewFilterSharpness() synced
                                                    per-frame to SetFilterSharpness()
  [100%] Runtime filter mode switching            — SyncScreenFXToBackend() in main loop
                                                    pushes filter mode changes to backend

================================================================================

  [100%] FBO pyramid creation                     — 5 levels (half-size each) + linear FBO

  [100%] Final composition pass shader            — 9-tap base + shoulder soft-clip +

  [100%] Bloom enable/disable                     — mScreenFX.mbBloomEnabled
  [100%] Bloom radius                             — mScreenFX.mBloomRadius
  [100%] Bloom direct/indirect intensity          — mScreenFX.mBloomDirectIntensity /
