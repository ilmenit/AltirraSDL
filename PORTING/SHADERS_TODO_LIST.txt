3. SCREEN EFFECTS — SCANLINES
================================================================================

  [  0%] Scanline mask texture generation         — Port VDDisplayCreateScanlineMaskTexture()
                                                    to GL texture. Raised cosine pattern,
                                                    regenerated on window resize.
  [  0%] Scanline fragment shader                 — Sample scanline mask, multiply with
                                                    frame color. Intensity uniform.
  [  0%] Scanline intensity control               — ATArtifactingParams::mScanlineIntensity
                                                    (0-100%) mapped to shader uniform

  Note: Current SDL3 build has GTIA software scanlines (SetScanlinesEnabled).
  Shader scanlines replace this with GPU-rendered scanlines that scale properly
  with the output resolution.
